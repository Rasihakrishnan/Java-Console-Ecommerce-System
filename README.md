# Java Console Application - Object Oriented Programming Project 💻

## 📌 Project Overview

This project is a complete console-based Java application developed with a strong focus on Object-Oriented Programming (OOP) principles and structured software design.

The application consists of approximately **1,681 lines of Java code** written as a single-file implementation. It was built to strengthen my understanding of developing larger applications, managing complex program flow, handling user interactions, and solving real-world programming challenges.

This project represents my journey from writing simple Java programs to designing a complete menu-driven console application.

---

## 🚀 Key Features

* Interactive menu-driven user interface
* User input handling and validation
* Multiple functionalities integrated into a single application flow
* Object-oriented design using classes and objects
* Encapsulation of data and behavior
* Exception handling for unexpected user inputs and runtime scenarios
* Conditional logic and iterative processing
* Organized methods to maintain readability and functionality
* Real-time console interaction
* Modular approach to problem-solving

---

## 🏗️ Application Flow

The overall execution flow of the application follows the structure below:

```
Application Start
        ↓
Display Main Menu
        ↓
Receive User Input
        ↓
Validate Input
        ↓
Process User Request
        ↓
Execute Business Logic
        ↓
Update / Retrieve Data
        ↓
Display Result to User
        ↓
Return to Menu or Exit Application
```

This flow helps maintain a smooth user experience while ensuring the application handles different operations efficiently.
